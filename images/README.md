# Carpeta de Imágenes - IPTV Pro

Esta carpeta contiene todas las imágenes utilizadas en el sitio web de IPTV Pro.

## Archivos de Imagen

### favicon.svg
- Ícono del sitio (favicon)
- Usado en la pestaña del navegador
- Dimensiones: 64x64px

### Pasos de Instalación
- **1.svg** - Paso 1: Descargar la aplicación
- **2.svg** - Paso 2: Instalar en tu dispositivo
- **3.svg** - Paso 3: Configurar credenciales
- **4.svg** - Paso 4: Verificar y disfrutar

Todos los archivos SVG son redimensionables y se adaptan a cualquier tamaño.

## Nota
Los archivos actuales son versiones SVG de ejemplo. 
Para producción, reemplazar con imágenes reales en formato WebP o PNG.

## Ubicación en el Código
Las imágenes se referencian en el HTML de la siguiente manera:
```html
<link rel="icon" href="images/favicon.svg" type="image/svg+xml">
<img src="images/1.svg" alt="Paso 1: Descargar" />
```
